Image with object fit : none


Image with object fit : cover : The image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit


Image with object fit : contain ; The image keeps its aspect ratio, but is resized to fit within the given dimension


Image with object fit : fill :This is default. The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit


Image with object fit : none

Image with object fit : scale-down: the image is scaled down to the smallest version of none or contain
